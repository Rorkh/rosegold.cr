    rosegold

    Minecraft client written in Crystal, following the botting rules of CivMC

    Use as a standalone client

    1. Install Crystal
    2. Install the Shards dependency manager
    3. Download the code: git clone git@github.com:grepsedawk/rosegold.cr.git
    4. Install the dependencies: shards install
    5. Build the executable: crystal build src/rosegold.cr
    6. Launch the client: ./rosegold
    7. Follow the instructions to log into your Minecraft account

    Use as a library

    1. Add the dependency to your shard.yml:

      dependencies:
  rosegold:
    github: grepsedawk/rosegold.cr

    2. Run shards install

    3. Import the module:

      require "rosegold"
